
CZ/
Basic Zip Browser
Tento program umožňuje procházet a prohlížet obsah ZIP souborů.

Funkce:
    Výběr ZIP souboru
    Zobrazení stromové struktury souborů v ZIPu
    Zobrazení informací o vybraném souboru

Použití:
    Spusťte program.
    Klikněte na tlačítko "Select ZIP" a vyberte ZIP soubor, který chcete procházet.
    Stromová struktura obsahu ZIP souboru se zobrazí v treeview.
    Vyberte soubor v treeview.
    Klikněte dvakrát na soubor pro zobrazení informací o vybraném souboru, nelze aplikovat na adresáře.

Požadavky:
    Python 3.7 nebo novější

ENG/
Basic Zip Browser

This program allows you to browse and view the contents of ZIP files.

Features:
ZIP file selection
Displaying a tree structure of files in the ZIP
Displaying information about the selected file

Usage:
Run the program.
Click the "Select ZIP" button and select the ZIP file you want to browse.
The tree structure of the ZIP file contents will be displayed in the treeview.
Select a file in the treeview.
Double-click a file to display information about the selected file. This does not apply to directories.
Requirements:

Python 3.7 or newer
